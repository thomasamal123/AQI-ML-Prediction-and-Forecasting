Available API Endpoints

The system also exposes REST APIs for integration and testing.

  • GET /api/features/pollutant – pollutant feature list
  • POST /api/predict/pollutant – current AQI prediction

Dataset scope: Delhi monitoring stations (2018–2019).
Forecasting is restricted to available historical data.
